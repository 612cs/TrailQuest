# Thesis 文档索引

`thesis/` 是论文资料的统一入口。旧的 `lunwen/`、`PPT/` 等目录已经收拢到这里，后续新增论文内容优先按下面的模块落位。

## 目录速查

### `chapters/`

论文 Markdown 分章草稿。

### `manuscripts/drafts/`

草稿、早期版本和未定稿内容。

### `manuscripts/revisions/`

修订链、版本迭代稿和处理中间稿。

### `manuscripts/final/`

最终稿、定稿和可直接交付的 Word 文档。

### `manuscripts/backups/`

Word 备份文件与历史备份。

### `figures/code-tables/`

论文第 5 章代码表格等 SVG 图。

### `figures/entity-diagrams/`

论文实体关系图源文件与导出图。

### `figures/feature-diagrams/`

论文功能图、流程图、架构图等。

### `figures/formulas/`

论文中的公式图与对应导出文件。

### `figures/placeholders/entities/`

实体占位图、表结构预览图等。

### `notes/`

AIGC、改写对照、说明笔记等辅助文档。

### `presentation/`

答辩 PPT、HTML 演示页和资源文件。

### `reports/`

检测报告、导出报告等附件。

### `scripts/`

论文处理、排版和生成脚本。
