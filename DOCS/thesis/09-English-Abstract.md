# English Abstract

With the rapid development of the digital economy, the growing awareness of public health, and the continuous advancement of smart tourism, outdoor activities such as hiking, mountaineering, and camping have become increasingly popular.

As user demand evolves, outdoor route platforms are expected to provide not only basic information retrieval, but also personalised recommendation, environmental awareness, intelligent interaction, and community sharing.

However, existing map tools and conventional tourism platforms still face problems such as fragmented information, insufficient data integration, weak recommendation capability, and limited support for risk-aware decision making in complex natural environments.

To address these issues, this paper studies the design and implementation of an outdoor route intelligent recommendation platform based on Vue, and further updates and refines the system in combination with the existing project.

The system adopts a front-end and back-end separation architecture.

The front end is built with Vue, TypeScript, Tailwind CSS, and DaisyUI to provide user interaction and page rendering.

The back end is implemented with Spring Boot 3, MyBatis-Plus, MySQL, Redis, and Spring Security to support business services, data management, and access control.

For intelligent interaction, the system follows the established technical route by introducing the DeepSeek large language model and the Function Calling mechanism to support natural language driven route retrieval and recommendation.

For environmental decision support, the platform combines Amap API, weather-related capabilities, and a Random Forest model encapsulated by Python Flask to provide probabilistic hints for outdoor landscape scenarios such as cloud sea and sunrise.

In addition, the platform organises a relatively complete business process around route publishing, community interaction, personal profile management, review management, and media upload.

Based on the existing thesis, the task document, the project proposal, and the current project implementation, this paper systematically discusses the research background, key technologies, requirement analysis, overall design, detailed implementation, and testing and optimisation of the platform.

The results show that the platform has practical value in route information organisation, user interaction experience, intelligent recommendation capability, and engineering completeness.

The thesis also retains the original optimisation conclusions on system performance and AI cost control, indicating that the platform has strong potential in both response efficiency and intelligent service cost reduction.

This research provides a meaningful engineering reference for the integration of smart tourism, outdoor services, and artificial intelligence in vertical application scenarios.

Keywords: Vue; outdoor route recommendation; intelligent interaction; DeepSeek; GIS; Random Forest
